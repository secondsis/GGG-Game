# ALL IS VOIDBOUND
ALL IS VOIDBOUND is a 2D tile-based puzzle game that follows a white rabbit looking for a carrot. You take turns, first playing as THE VOID and destroying objects blocking the path, then playing as the white rabbit with the goal of reaching the carrot. 

# THE GAME JAM
This was made for the GGG game jam! The theme was "The Void" for the Girls Got Game game jam. 

# INSTRUCTIONS
Go to https://secondsis.itch.io/all-is-voidbound and hit "Run game". That's it!

As for controls, it is WASD or the arrow keys. The game does NOT have mobile compatibility. Press "R" to restart a level. 

# MECHANICS
All objects are consume-able by THE VOID, with the exception of the Carrot. 


* <b>Wall</b>: The rabbit cannot pass through these.
* <b>Key</b>: Able to be picked up by the rabbit
* <b>Locked Door</b>: Requires the rabbit to stand next to the door with a key to be destroyed, or THE VOID can also destroy it
* <b>Pushable Crate</b>: The rabbit is able to push these, including multiple crates. 
* <b>Ghost</b>: Kills the rabbit and restarts the level upon the rabbit touching it. It will also use up x2 VOID points. 
* <b>Planks</b>: The rabbit can walk across these, but THE VOID will still destroy it upon touch
